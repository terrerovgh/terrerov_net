Bienvenido a Terrerov Network

Esta es una página de prueba que demuestra el funcionamiento correcto de la infraestructura de red.

Servicios Activos:

  • DNS Server (Bind9) - Resolución de nombres de dominio
  • Traefik - Proxy inverso y gestión de certificados SSL
  • Nginx - Servidor web (¡estás viendo esta página!)
  • PostgreSQL - Servidor de base de datos

Si puedes ver esta página, significa que el servidor web Nginx está funcionando correctamente a través de Traefik con SSL.
